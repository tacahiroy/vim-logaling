INTRODUCTION					*loga-intro*

*vim-logaling* is a simple wrapper of logaling-command.
It requires Vim 7.2 or later.

This plugin only works if 'compatible' is not set.
{Vi does not have any of these features.}


==============================================================================
COMMANDS					*loga-commands*

						*loga-:Loga*
:Loga [command] [args]  Run any logaling-command whatever you want.
                        Similar to :!loga [command] [args]

						*loga-:Ladd*
:Ladd {source term} {target term} [note]

						*loga-:Ldelete*
:Ldelete {source term} [target term] [--force] [args]

						*loga-:Lhelp*
:Lhelp {command}

						*loga-:Llist*
:Llist

						*loga-:Llookup*
:Llookup {term} [args]

						*loga-:Lshow*
:Lshow  [args]

						*loga-:Lupdate*
:Lupdate {source term} {target term} {new target term} [note] [args]


==============================================================================
SETTINGS					*loga-settings*
These variables control behaviour of the |vim-logaling|.


						*g:loga_result_window_hsplit*  >
  g:loga_result_window_hsplit = 1
<	This variable controls split direction of a result window.
	If it's 1, window is split horizontal. If not, vertical.

						*g:loga_result_window_size*  >
  g:loga_result_window_size = 5
<	This variable controls size of a result window.
	Only an integer can be specified.
